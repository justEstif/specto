Specto

Your digital life is fragmented across dozens of platforms. Specto pulls it all together — one dashboard for everything you watch, listen to, and read.

GitHub Self-Host

What It Does

Unified Media Feed

Pull consumption history from Spotify, YouTube, Netflix and more into a single timeline.

Auto-Tagging

Content is classified by topic, genre, and mood using platform APIs and LLM enrichment.

Consumption Insights

See time spent, topic distribution, platform breakdown, and trends over time.

Shareable Profiles

Generate a public page of your media diet — like Spotify Wrapped, but for everything.

Supported Platforms

Connect via OAuth or file import. Each platform is a plugin — disable what you don't use.

  • Spotify
  • YouTube
  • Netflix
  • Prime Video
  • TikTok
  • Podcasts
  • RSS / Blogs

Tech Stack

Go + Chi · PostgreSQL + sqlc · Templ + HTMX · DaisyUI + Tailwind CSS

No JavaScript framework. No build step for the frontend. Server-rendered HTML with progressive enhancement.

Self-Host in Minutes

git clone https://github.com/justestif/specto
cd specto
docker-compose up -d
mise install && mise run setup
mise run build
./bin/app

See the self-hosting guide for full configuration (env vars, OAuth setup, reverse proxy).

How It Works

  1. Connect your platforms via OAuth or upload an export file
  2. Sync pulls your history into a normalized schema
  3. Enrich auto-tags content using Last.fm, TMDB, AniList, and LLMs
  4. Explore your consumption through the insights dashboard
  5. Share a curated public profile of your media diet
